# Athanor

Athanor parses your `.github/workflows/` YAML files and executes them locally with a
terminal UI. Test your CI pipelines without pushing — pick a workflow, watch jobs run in
real time.

## How It Works

1. Athanor discovers workflow files in `.github/workflows/`
2. A TUI lets you pick which workflow to run
3. Jobs are topologically sorted — independent jobs run concurrently
4. `run:` steps execute via bash with live-streamed output
5. A split-panel view shows the job/step tree alongside log output

## Why Athanor?

- **Fast feedback** — no push/wait/check cycle, run workflows on your machine
- **Visual** — a terminal UI shows progress, not raw shell output
- **Simple** — single binary, no daemon, no Docker, no configuration files beyond the workflow YAML you already have
- **Future-proof** — the long-term goal is full microVM isolation via CloudHypervisor, but local execution works today
